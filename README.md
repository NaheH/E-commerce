# E-commerce React & Go

Ce projet est une application de **site e-commerce** développée en groupe, avec **React** pour le frontend et **Go** pour le backend. L'application permet de naviguer parmi des produits, d'ajouter des articles au panier, de passer des commandes, et de gérer un compte utilisateur.

## Objectif du Projet

L'objectif de ce projet est de créer une plateforme e-commerce fonctionnelle avec les principales fonctionnalités de gestion d'inventaire, panier d'achat, et de commande. L'accent a été mis sur la performance et l'expérience utilisateur, en utilisant des technologies modernes côté frontend et backend.

### Fonctionnalités Clés

  - Affichage de produits avec des informations détaillées.
  - Interface de gestion du panier en temps réel.
  - Formulaires de connexion, inscription, et profil utilisateur.
  - Page de validation de commande avec formulaire d'adresse et de paiement.
  - API RESTful pour la gestion des produits, utilisateurs, et commandes.
  - Gestion des sessions et des jetons d’authentification pour la sécurité.
  - Connexion à une base de données (par exemple, PostgreSQL) pour stocker les informations des produits, utilisateurs, et commandes.
  - Paiement google sécurisé.
    
## Prérequis

Avant de commencer, assurez-vous d'avoir installé les éléments suivants :
- **Node.js** et **npm** (pour le frontend React)
- **PostgreSQL** ou autre base de données compatible pour stocker les informations
